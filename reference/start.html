インストール

1. "qrone-x.y.z.jar" をダウンロードしよう！

まず。最新の qrone.zip をダウンロードしましょう。
また、Java 6 がインストールされていない場合、インストールしてください.

また、開発版を入手したいなどの場合には、github のリポジトリを参照してもよいでしょう。

2. 動かしてみよう！

QrONE は、１ファイルで起動出来るウェブサーバーです。

インストールも qrone.zip を解凍するだけでよく、解凍したフォルダにある qrone-x.y.z.jar だけで起動できます。残りのファイルはサンプルのソースコードや、ライセンスファイルなどです。

Windows なら "qrone-x.y.z.jar" をダブルクリックして起動します。

一切の設定はありませんので、まず起動してみましょう。起動すると下のようなウインドウが表示されます。これで 9601 ポートでウェブサーバーが起動できました。また、終了したいときはウインドウを閉じるだけです。



もちろん、コマンドラインからも起動可能で、プロダクションで利用するための数多くのオプションが用意されています。

また、qrone.jar は、Servlet として配置することも可能な為、既存の環境を壊すことなく、Tomcat、Jetty (QrONE はデフォルトで Jetty を利用しています) などの Java サーバーや Google AppEngine で動作させることも可能です。

3. スクリプトを実行してみよう！

QrONE は、デフォルトで jar ファイルのあるフォルダを、ドキュメントルートとして参照するようになっています。

メモ帳などを利用して "hello.js" という名前のファイルを qrone.jar のあるフォルダに下の内容で作ってみましょう。

document.write("Hello JavaScript World!");

また、qrone.jar はこのファイル１つだけで動作しますので、好きな場所に移動して簡単に利用できます。いつもウェブサイト開発をしているフォルダがあれば、そこに qrone.jar を移動して起動してみましょう。

4. 表示してみよう！

以上で完了です。
http://localhost:9601/test.js にアクセスすると Hello JavaScript World! と表示されましたね。
